26 /* Local copy propagation and dead code elimination within a single action function.
27  This pass is designed to be run after all declarations have received unique
28  internal names. This is important because the locals map uses only the
29  declaration name, and not the full path.
30 
31  Requires expression types be stored inline in the expression
32  (obtained by running Typechecking(updateProgram = true)).
33  Requires that all declaration names be globally unique
34  (obtained by running UniqueNames)
35  */
